Question-1 : What are the different ways to select an element in the DOM ?


Answer : There are different ways to select an element in the DOM.


  1. 1. document.getElementById("id") ; Selects an element by its ID.
  2. 2. document.getElementsByClassName("class") ; Selects an element by Class Name.
  3. 3. document.getElementsByClassName("tag") ; Selects an element by Tag Name.
  4. 4. document.querySelectorAll("selector") ; Selects all matching elements using a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?


Answer :

1. innerHTML: Returns or sets the HTML content inside an element, including any HTML tags.
2. innerText: Returns or sets only the visible text inside an element, ignoring hidden elements and extra spaces.
3. textContent: Returns or sets all text inside an element, including hidden text, without parsing HTML.

Question-3: What is event delegation in the DOM ?


Answer :

Event Delegation in the DOM is a technique where a single event listener is added to a parent element to handle events on its child elements, even if they are added dynamically. It works by taking advantage of event bubbling, where events propagate up from the target element to its ancestors.

Question-4: What is event bubbling in the DOM?


Answer :

Event Bubbling in the DOM is a mechanism where an event triggered on a child element propagates up through its parent elements until it reaches the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?


Answer :

1. Creating Element : By using 'document.createElement()' command.

2. Adding Element : By using appendChild() or append() to add it inside a parent element.

3. Removing Element : By using removeChild() or remove() command.
